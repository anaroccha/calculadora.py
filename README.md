# calculadora.py
Este programa simula uma calculadora simples que recebe dois números e um operador
informado pelo usuário, realizando a operação correspondente.

Lógica:
1. O usuário informa dois números e o operador que quer utilizar.
2. Os valores são convertidos para o tipo float.
3. O programa realiza a operação.
4. O programa verifica se o usuario escolheu um operador que é valido (+-/*) e se escolheu mais de um operador.
5. O resultado é exibido no terminal.
6. O programa trata possíveis erros, como valores inválidos.

Aprendizado:
- Uso da função input()
- Conversão de tipos (string para float)
- Operações matemáticas básicas
- Estruturas condicionais (if/elif)
- Tratamentos de erros com Try/Except 
- Exibição de resultados com print()
